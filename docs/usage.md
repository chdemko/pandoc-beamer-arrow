Usage
=====

To apply the filter, use the following option with pandoc:

~~~shell-session
$ pandoc --filter pandoc-beamer-arrow
~~~

Explanation
-----------

*pandoc-beamer-arrow* creates arrows between text elements for
beamer presentation.
